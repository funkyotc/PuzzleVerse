# Proposed Games for PuzzleVerse

1. 2048
Type: Sliding tile puzzle.
Mechanics: Swipe grid. Combine identical numbers. Reach 2048 tile.
Pros: Simple logic. Highly addictive. Easy UI implementation.
Cons: Very common.
2. Nonogram (Picross)
Type: Picture logic puzzle.
Mechanics: Use number clues on rows and columns. Color grid cells. Reveal hidden picture.
Pros: Fits PuzzleVerse theme. Visually rewarding. Scalable difficulty (5x5 to 15x15).
Cons: Complex state management for board.
3. Minesweeper
Type: Logic puzzle.
Mechanics: Uncover safe tiles. Flag bombs. Use number clues.
Pros: Classic game. Easy rules. Good for speed/time tracking.
Cons: Mobile tapping precision can be tricky for larger boards.
4. Block Puzzle (Tetris style, but static)
Type: Block placement.
Mechanics: Drag given block pieces onto 10x10 board. Clear full rows or columns.
Pros: Endless gameplay. Relaxing mode possible.
Cons: Needs custom drag-and-drop mechanics (though already partially solved by Bonza/Shapes).
5. Kakuro / KenKen
Type: Math logic puzzle.
Mechanics: Like Sudoku, but with addition. Fill numbers to sum up to target clues.
Pros: Great for math lovers. Fits nicely next to Sudoku.
Cons: Harder to generate puzzles automatically than Sudoku.
6. Flow Free (Connect Dots)
Type: Pathfinding logic.
Mechanics: Connect matching colored dots on grid with pipes. Pipes cannot cross. Fill entire grid.
Pros: Simple UI. Colorful. Very popular mobile mechanic.
Cons: Complex puzzle generation algorithm needed.
